AgentSystem

A comprehensive .NET-based framework for building and orchestrating autonomous agents with web and mobile interfaces.

Get Started View on GitHub

Key Features

Multi-Modal Analysis

Evaluates agent outputs across different modalities including text, visual, and audio.

Plugin Architecture

Extensible plugin system for adding new capabilities and integrating with external services.

Template-Driven Design

Uses .prompty files to define agent behaviors and capabilities with a flexible templating system.

Cross-Platform UI

Web and mobile interfaces built with MAUI and Telerik UI components for rich user experiences.

Distributed Evaluation

Comprehensive metrics for assessing agent performance across multiple dimensions.

Aspire Integration

Built on .NET Aspire for cloud-native microservices architecture with monitoring and resilience.

Architecture

Microservices Design

AgentSystem is built using a modern microservices architecture with .NET Aspire:

  • API Gateway: Central entry point for all client interactions
  • Core Services: Fundamental agent processing capabilities
  • Analysis Services: Specialized processing for different evaluation dimensions
  • Infrastructure: Shared resources like caching, database, and messaging

The modular design allows for independent scaling and deployment of components.

Project Structure

The solution is organized into several key projects:

  • AgentSystem.Core: Core framework, interfaces, and agent implementations
  • AgentSystem.Plugins: Plugin system for extending agent capabilities
  • AgentSystem.Web: Blazor-based web interface with MAUI integration
  • AgentSystem.Mobile: MAUI-based mobile application
  • AgentSystem.AppHost: Aspire host for microservices architecture
  • AgentSystem.ServiceDefaults: Default service configurations

Client Integration

Example of client-side integration with the AgentSystem API:

// Client-side integration
using var client = new HttpClient();
client.BaseAddress = new Uri("https://agent-api.example.com/");

var assessmentRequest = new AssessmentInput
{
    MultiModalOutput = captureMultiModalOutput(),
    EmbodiedCognitionOutput = captureEmbodiedOutput(),
    // ... other outputs
};

var response = await client.PostAsJsonAsync(
    "api/assessment", 
    assessmentRequest
);
var result = await response.Content
    .ReadFromJsonAsync<AssessmentResult>();

Getting Started

Prerequisites

  • .NET 9.0 SDK
  • Visual Studio 2022 or later / VS Code
  • Docker Desktop (for local development)
  • SQL Server (local or remote)

Installation

# Clone the repository
git clone https://github.com/TooensureAI/AgentSystem.git
cd AgentSystem

# Restore dependencies
dotnet restore AgentSystem.sln

# Build the solution
dotnet build AgentSystem.sln

Running the Applications

# Run the web application
dotnet run --project AgentSystem.Web/AgentSystem.Web.csproj

# Run the mobile application (requires emulator/device)
dotnet run --project AgentSystem.Mobile/AgentSystem.Mobile.csproj

# Run the AppHost (for microservices architecture)
dotnet run --project AgentSystem.AppHost/AgentSystem.AppHost.csproj

Configuration

Edit the appsettings.json file in the AppHost project to configure your environment:

{
  "Aspire": {
    "Dashboard": {
      "Enabled": true,
      "Port": 18888
    }
  },
  "Assessment": {
    "ValidationThresholds": {
      "EthicalScore": 0.8,
      "SemanticCoherenceScore": 0.7
      // Other settings...
    }
  }
}
